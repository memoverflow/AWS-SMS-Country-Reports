# AWS SMS 各国发送要求研究报告

> 调研日期：2026年4月7日
> 数据来源：AWS 官方文档、各国电信监管机构资料、运营商文档

## 概述

本仓库包含 **45 个国家/地区** 在 AWS End User Messaging SMS（前身为 Amazon Pinpoint SMS）上的短信发送要求研究报告，涵盖注册流程、合规要求、消息模板、定价参考等内容。

## 快速参考

### 注册难度分类

- **无需注册（16 国）**：DE, IT, ES, AT, PT, RO, FI, NL, SE, JP, NO, DK, CZ, HU, IL, HK
- **需要注册（13 国）**：US, GB, SG, IE, CN, AU, TH, TR, SA, AE, CA, FR, MX
- **功能极度受限（4 国）**：KR, CH, GR, AR

### 高风险国家

| 风险 | 国家 | 原因 |
|------|------|------|
| 极高 | KR | AWS 几乎不支持 |
| 高 | CN | 模板必须预注册 |
| 高 | SG, IE | 未注册显示 "LIKELY-SCAM" |
| 高 | US | TCPA 违规每条 $500-$1,500 |

## 免责声明

本报告基于 2026 年 4 月公开资料编写。各国电信法规和 AWS 服务支持可能随时变更，实际使用前请在 AWS 控制台确认最新信息，并咨询当地法律顾问以确保合规。
